# Schema Ownership

- Status: Reserved; ORKS-0202 preparation in progress

This directory is reserved for JSON Schemas derived from the exact
[pinned standard input](../SUPPORTED-STANDARD.md), using JSON Schema Draft
2020-12 under approved ORKS-0202 work.

It currently contains no schema, compiler output, or executable standard
interpretation. Task-local validation dependencies and qualification data are
present elsewhere in the repository. Future schemas cannot create, weaken,
reinterpret, or silently repair normative behavior absent from the pinned
source tree.

Return to the [documentation index](../docs/README.md).
