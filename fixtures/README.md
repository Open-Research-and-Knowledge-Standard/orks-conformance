# Fixture Ownership

- Status: Reserved for later approved work

This directory is reserved for future positive, negative, boundary, hostile,
canonical-byte, locator, mutation, and compatibility fixtures derived from the
exact [pinned standard input](../SUPPORTED-STANDARD.md).

It currently contains no executable fixture payload or fixture-layout
contract. Future fixtures require explicit provenance and expected outcomes,
follow the [fixture policy](../docs/fixture-policy.md), remain non-fetching by
default, and do not fetch source content during deterministic validation.

Return to the [documentation index](../docs/README.md).
